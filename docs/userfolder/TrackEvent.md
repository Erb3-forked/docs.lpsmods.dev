# TrackEvent

The track event returned by trackcommand

## Parameters

| Name     | Type            | Description                 |
| -------- | --------------- | --------------------------- |
| `member` | zipfile.ZipInfo | The Zip.Info or filename    |
| `count`  | int             | The current member of total |
| `total`  | int             | The total number of members |
